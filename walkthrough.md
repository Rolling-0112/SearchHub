## 📱 移动端功能 (Mobile Features)

### 1. PWA 安装 (像 App 一样使用)
*   **iOS (Safari)**: 点击底部分享按钮 -> "添加到主屏幕"。
*   **Android (Chrome/Edge)**: 点击菜单 -> "安装应用" 或 "添加到主屏幕"。
*   **效果**：安装后拥有独立图标、全屏体验，且无浏览器地址栏干扰。

### 2. 针对性优化
*   **小红书智能降级**：
    *   **电脑端**：直接打开小红书网页。
    *   **手机端**：**自动唤醒小红书 APP** 进行原生搜索。
*   **哔哩哔哩直接唤醒** (V3.4)：
    *   手机端搜索 Bilibili 时，将直接唤醒 **哔哩哔哩 APP**。
*   **性能模式**：手机上自动关闭高消耗的毛玻璃特效，确保滑动丝滑。
*   **深浅色主题切换** (V3.5)：
    *   右上角点击 ☀️/🌙 图标即可切换。
    *   系统会自动记住您的选择。
*   **防误触**：放大了图标点击区域，并添加了点击的高亮反馈 (Active State)。

## 🛠️ 项目维护 (Maintenance)

### 自动部署 (GitHub Actions)
您现在只需要在本地修改代码并推送：
```bash
# 只要这一步，三个平台(GitHub/Vercel)都会自动更新
git push origin main
```
*   **GitHub Pages**: 约 2-3 分钟自动生效。
*   **Vercel**: 约 1 分钟自动生效。

### 数据库 (Firebase Firestore)
*   数据存储在云端，目前处于测试模式。
## 🐛 故障排查 (Troubleshooting)

### 常见问题
如果您的更新没生效（例如搜 B站 还是跳网页），请尝试：
1.  **强制刷新** (Chrome: Ctrl+F5)
2.  **清空缓存** (手机设置 -> 应用 -> 浏览器 -> 清除数据)
3.  **重新添加到桌面** (PWA 用户)
